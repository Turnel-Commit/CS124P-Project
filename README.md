# CS124P Project

Welcome to the CS124P Project repository! This project contains machine learning and computer vision applications developed for coursework and research purposes.

## Project Overview

This repository contains implementations of various computer vision and data analysis projects, including face recognition and rice disease measurement systems.

## Directory Structure

### 📁 Face Recognition
Machine learning implementation for facial recognition tasks. This directory contains models, training scripts, and utilities for face detection and recognition.

**Features:**
- Face detection capabilities
- Facial feature extraction
- Recognition model training and evaluation
- Preprocessing utilities for image data

### 📁 RiceMeasurement Diseases
A specialized system for measuring and identifying diseases in rice crops. This project utilizes computer vision techniques to detect and classify various rice plant diseases.

**Features:**
- Disease detection in rice plants
- Image classification models
- Data analysis and visualization
- Model evaluation metrics

## Getting Started

### Prerequisites
- Python 3.7+
- Libraries: OpenCV, TensorFlow/PyTorch, NumPy, Pandas
- Jupyter Notebook (optional, for exploratory analysis)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Turnel-Commit/CS124P-Project.git
cd CS124P-Project
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

### Usage

Navigate to the specific project directory and follow the instructions in that project's documentation.

#### Face Recognition
```bash
cd "Face Recognition"
# Follow the project-specific README or notebook instructions
```

#### Rice Disease Measurement
```bash
cd "RiceMeasurement Diseases"
# Follow the project-specific README or notebook instructions
```
